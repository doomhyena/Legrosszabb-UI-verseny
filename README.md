# Bláthy nap 2026 – „Legrosszabb UI verseny”

## Projekt leírás
Ez a projekt egy szándékosan **rossz felhasználói élményt nyújtó bejelentkezési felület**, amely egy képzeletbeli közösségi média alkalmazáshoz készült.

A cél az volt, hogy a lehető **legidegesítőbb, leglassabb, legértelmetlenebb és leginkább használhatatlan** login folyamatot hozzuk létre – természetesen humoros formában.

--

## Technológiák

- HTML / CSS / JavaScript

##  Használat

1. Nyisd meg az alkalmazást
2. Próbálj meg bejelentkezni
3. Gondold át az életed döntéseit
4. Add fel

## Design célok

A projekt során az alábbi UX alapelveket **szándékosan megszegtük**:

- Konzisztencia
- Felhasználóbarát visszajelzés
- Egyszerűség
- Gyorsaság
- Logikus folyamat
